## 🚀 快速开始

### 1. 本地克隆
```bash
git clone https://github.com/Xie-Dongshu/Dongshu_blog.git
cd Dongshu_blog
```

### 2. 创建新笔记
使用 `NOTES_TEMPLATE.md` 作为模板创建新笔记：
```bash
cp NOTES_TEMPLATE.md notes/[分类]/your-topic.md
```

### 3. 编写和提交
```bash
git add .
git commit -m "添加：[笔记标题]"
git push origin main
```

## 📖 如何使用笔记模板

每篇笔记都应该遵循 `NOTES_TEMPLATE.md` 的格式，包含：
- 标题和日期
- 摘要
- 主要内容
- 关键要点
- 参考资源
