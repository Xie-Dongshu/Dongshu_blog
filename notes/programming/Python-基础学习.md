## 📖 主要内容

### 第一部分：Python介绍

Python是一种高级编程语言，具有简洁易读的语法，广泛应用于Web开发、数据分析、人工智能等领域。

### 第二部分：基本语法

```python
# 注释示例
print("Hello, Python!")  # 输出语句

# 变量赋值
name = "Dongshu"
age = 25

# 条件语句
if age >= 18:
    print("成年人")
else:
    print("未成年")

# 循环
for i in range(5):
    print(f"第 {i} 次循环")
```

### 第三部分：数据类型

```python
# 整数
integer = 42

# 浮点数
floating = 3.14

# 字符串
string = "Hello, World!"

# 列表
my_list = [1, 2, 3, 4, 5]

# 字典
my_dict = {"name": "Dongshu", "age": 25}

# 元组
my_tuple = (1, 2, 3)
```

---

## 💡 关键要点

1. **缩进很重要**: Python使用缩进表示代码块
2. **动态类型**: 不需要提前声明变量类型
3. **强大的库**: Python有丰富的第三方库
